1. Cos’è ReactJS?
React è una libreria che permette di scrivere interfacce utente e logiche del sito, in modo modulare, ovvero scomponendo il sito
in piccoli componenti, riutilizzabili e facilmente manutenibili. Si suddivide in 3 sezioni: l'index.html che viene caricato in principio
ma in cui non vi è altro che un div vuoto con id=root dove verrà iniettato il codice con react successivamente. oltre a ciò si può trovare
collegamenti a script essenzioakli o fogli di stile.
la seconda sezione index.js, si può considerare come il "motore" di react che importa e carica
i pacchetti relativi al suo funzionamento(dalla cartella node_modules disponibili dopo l'installazione),importa il file js dove creo i componenti
e file di stile. poi vi è la parte dove punto il root e inserisco il dom virtuale di react. la terza sezione sono appunto l'interfaccia e i componenti
creati, con il return di una funzione(scritta come html ma sotto il cofano funziona come javascript).
